# Ruboty::SlideMe

show today's slide from [Slidegate](http://slidegate.herokuapp.com/)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-slide_me'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-slide_me

## Usage

    @ruboty slide me
